🛡️ Privacy Commitment

The EECOL Wire Tools Suite v0.8.0.4 prioritizes enterprise-grade privacy and data security. Your data remains under your control within trusted networks, without reliance on any third-party backend services.

📊 No External Data Collection

  • ✅ No personal information collected (name, email, location, IP address)
  • ✅ No analytics or tracking technologies
  • ✅ No third-party data sharing or backend services
  • ✅ Complete data sovereignty and network independence

💾 Local Data Storage

Enterprise Storage Policy

  • ✅ All data stored in browser IndexedDB with full data integrity
  • ✅ Data accessible only within your local network boundaries
  • ✅ Clearing browser data will erase local copies only
  • ✅ Comprehensive data export/import and backup capabilities

What We Store Locally:

  • 📋 Wire cutting records (order numbers, measurements, cutter information)
  • ⚙️ User preferences (calculator settings, measurement units)
  • 📦 Complete inventory tracking with approval workflows
  • Advanced analytics and reporting data

🔒 Security Measures

Technical Security

  • ✅ Content Security Policy (CSP) prevents external script injection
  • ✅ Subresource Integrity (SRI) verifies external CDN libraries
  • ✅ HTTPS-only external resources (Tailwind CSS, Google Fonts)
  • ✅ IndexedDB database with transaction integrity and rollback
  • ✅ Service worker isolation for offline functionality
  • ✅ Progressive Web App manifest for native-like security

Enterprise-Grade Cryptography

  • ✅ WebRTC protocols with TLS encryption at all layers
  • ✅ Cryptographic hashing for data integrity verification
  • ✅ Secure random number generation for encryption keys
